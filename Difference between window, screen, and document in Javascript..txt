 Difference between window, screen, and document in Javascript.

 window
   -Window is the execution context and global object for that context's JavaScript
   -Each browser tab has its own top-level window object

 Document
   -Document contains the DOM, initialized by parsing HTML
   -Each window object has a document object to be rendered

 Screen
   -Screen describes the physical display's full screen.
   -The window object also has a screen object with properties describing the physical display such as screen properties width and height.
